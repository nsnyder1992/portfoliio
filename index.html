Parameter Extractor

Description: I was asked to develop software to connect to a PLC (Programable Logic Controller) and download/upload parameters, of the current recipes. Using python, wxpython, and Ethernet/IP (a common server/client Industrial Communication protocol) python library, I was able to accomplish this task. I setup a database, so that admins could login and add/remove certain parameters or even UDTs (classes in Ladder Logic programing).I also made a feature for field-service engineers (FSEs) to be able import/export parameters to CSV format. This came in handy when a operator would mess with the recipe and couldn't get it to work again, the FSE could just load the last working recipe!
